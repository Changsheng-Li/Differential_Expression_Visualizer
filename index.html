Hey there! Welcome 👋

This is a little all-in-one manual for poking around this webpage tool.

Before You Start

A few things to have ready:

  • Your data as a TXT, CSV or TSV file with at least these columns: a fold-change column (e.g. logFC), a p-value or probability from BASiCS MCMC column, and an entry name / gene symbol column.
  • You only need a browser to open this tool, but by the time you read this I believe you have already opened it :). Although for the Enrichment, Network tabs you'll need an internet connection — they call public APIs (g:Profiler, STRING) to do their job.
  • You don't need a running R-session or python environment. But you do need R if you want to do GSEA later.
  • All changes are real-time. The recommended workflow is: Volcano Plot first → then hop to enrichment analysis tab to import list before going to any other tab. The volcano is where your gene lists come from! And import gene lists step gives list to all other analysis.

1 — Volcano Plot

Your starting point. Upload data, tweak cutoffs, see what lights up.

1
Upload or drop your TXT/CSV/TSV and map your column names (logFC, P-value, Gene label) if they are not exactly the same as the default names. Be aware they are case-sensitive.
2
Play with FC cutoff, p-value cutoff, and cutoff mode until it feels right.
3
Hit Generate Plot. Hover dots for info, or type gene names in Highlight Genes to spotlight them.
4
Change colors if you want (defaults are color-blind friendly). Save as PNG when you're happy.
5
If you choose to export the table to CSV, there will be an extra column in the table to indicate the regulation direction according to the cutoff you set anew.

2 — Enrichment Analysis

Find out what pathways your genes are up to, powered by g:Profiler. Needs internet.

First things first: Make sure you have gene lists ready! Go to the Gene Lists section in this tab's sidebar and click Import Genes from Volcano to pull them in from your volcano plot current cutoff.
1
Pick your databases (GO, KEGG, Reactome, etc.) and gene source in the sidebar.
2
Click Run Enrichment Analysis and watch the dot plot appear. Hover for details.
3
Save the plot as PNG or grab the results as CSV.
Bonus: The "Uploaded Results" sub-tab lets you visualize results from external tools like clusterProfiler too.

3 — Network Analysis

See how your proteins talk to each other, courtesy of STRING. Needs internet.

First things first: Head over to the Enrichment Analysis tab first if you skipped it. And click Import Genes from Volcano so your gene lists are loaded.
1
Pick your organism, tweak the confidence slider and max genes.
2
Hit Run Network Analysis. Drag nodes around, scroll to zoom, hover for details.
3
Export as PNG, Interactions CSV, or Node Table CSV.
Good to know: Bigger nodes = more connections. Thicker edges = higher confidence. Colors follow your volcano color scheme.

4 — GSEA

Get a ready-made R script for fgsea, or upload your own GSEA results to visualize.

First things first: Head over to the Enrichment Analysis tab first if you skipped it. And click Import Genes from Volcano — the R script generator needs those gene lists.
1
Pick organism and MSigDB category, then click Generate R Script.
2
Copy the script or download it, run it in R/RStudio, and save the output CSV (in working directory by default, or specify a path).
3
You should have plot done in R session already it it found any positive hits. Or you can upload the fgsea results CSV back here, click Plot Bubble Plot, and export.

Tips & Tricks

  • Colors stay in sync — change them in the Volcano tab and everything else follows along.
  • Everything is exportable — every tab has PNG and CSV downloads.
  • All plots are interactive — hover for tooltips, and the network supports drag & zoom too.
  • This page is still under construction by Changsheng Li @ UMC Utrecht. SO there might be bugs. Let me know if you encounter any!
