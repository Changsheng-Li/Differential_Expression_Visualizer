# Differential_Expression_Visualizer
A webpage tool for DE analysis result that doesn't need any backend R/Python running session

## Changelog

- **v1.0.2** (2026-03-25) — Column mapping now uses dropdown menus populated from the uploaded file, with first-value previews and auto-detection of common column names.
- **v1.0.1** (2026-03-20) — Added Excel (.xlsx/.xls) file support and comma-as-decimal-separator option for European locale data.
- **v1.0.0** (2026-03-20) — Initial release with Volcano Plot, Enrichment Analysis (g:Profiler), Network Analysis (STRING), and GSEA tabs.
